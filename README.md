## Portfolio map
- [playwright-tests](https://github.com/rmpetrov/playwright-tests): UI + API framework + CI + reports
- `sdet-toolbox` (this repo): compact showcase with CI, published reports, and mini-projects
- [mini-projects/ui-playwright-smoke](mini-projects/ui-playwright-smoke/README.md): Playwright + Pytest UI smoke suite
- [mini-projects/api-requests-pytest](mini-projects/api-requests-pytest/README.md): API checks with Requests + Pytest fixtures and schema/key-field validation
- [mini-projects/contract-testing](mini-projects/contract-testing/README.md): OpenAPI + JSON schema validation
- [mini-projects/data-quality-tests](mini-projects/data-quality-tests/README.md): Data validation with Pandera on CSV samples
- [mini-projects/performance-k6-or-locust](mini-projects/performance-k6-or-locust/README.md): Locust smoke load test skeleton

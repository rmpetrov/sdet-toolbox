# Security policy

## Supported versions
- `main` branch: supported
- Release branches/tags: best-effort only

## Security scope
This repository contains test code and sample fixtures. Do not commit production secrets, real credentials, or private customer data.

## Reporting a vulnerability
Email security concerns to `rpetrovqa@gmail.com` with:
- impact summary
- reproduction steps or proof-of-concept
- affected paths/components

Acknowledgement target is within 5 business days on a best-effort basis.
